# SMU_Team_project

## 상명대학교 18년도 2학기 경영빅데이터 분석 **3조** 팀프로젝트

주제: 소셜데이터와 영화흥행도 간의 상관관계 분석(을 통한 성공적 마케팅 전략 제언)

분석 도구: 파이썬(자료 수집, 예측모델 수립) / R(데이터 정제 및 전처리) / Qlik or tableau(데이터 시각화)

사용 데이터: 
- 소셜데이터 (트위터, 페이스북 등 SNS채널 크롤링)
- 영화진흥위원회 ( 누적 관객수, 매출)
